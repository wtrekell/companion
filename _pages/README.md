# Core Site Pages

Essential static pages that form the primary navigation and informational backbone of the Syntax & Empathy Companion.

## Summary

This directory contains the foundational static pages for the Syntax & Empathy Companion website, including "About," "Articles," "Projects," and "Resources." These pages serve as crucial entry points, guiding visitors through the site's main content areas and providing an overview of its mission and offerings. They are designed to help users quickly understand the site's structure and locate relevant information on design leadership, AI collaboration, and technical workflows.

## About This Content

The `_pages` directory is central to the user experience of the Syntax & Empathy Companion. It hosts the top-level static pages that define the site's primary navigation and content categories. Each `.md` file within this directory corresponds to a distinct section of the website, employing a `single` layout and rich front matter to provide clear titles, descriptions, and permalinks. This content is designed for all site visitors, serving as a comprehensive introduction to the site's purpose, the topics it covers, and the types of content available. These pages articulate the site's commitment to exploring the intersection of technical precision and human-centered design, anchoring the broader context of articles, projects, and resources found elsewhere in the repository.

## Key Topics

-   **About Syntax & Empathy:** Details the philosophy, mission, and scope of the Syntax & Empathy Companion, focusing on design leadership, AI, and technical workflows.
-   **Articles:** An index page for various thoughts and insights on design leadership, Git workflows, AI collaboration patterns, and technical documentation.
-   **Projects:** Showcases a collection of creative experiments and applications developed using AI tools and innovative workflows.
-   **Resources:** A curated collection of useful tools, templates, code patterns, and educational materials relevant to AI, design systems, and technical documentation.

## Content Structure

This directory contains:

-   **`about.md`** - The "About" page, providing an an overview of the Syntax & Empathy Companion's purpose and vision, exploring technical precision and human-centered design.
-   **`articles.md`** - The main "Articles" index page, serving as a gateway to all published articles and thoughts on design leadership, Git workflows, and AI collaboration.
-   **`projects.md`** - The "Projects" index page, showcasing creative experiments with AI tools and applications.
-   **`resources.md`** - The "Resources" index page, offering a curated collection of useful tools, templates, and resources for working with AI and design systems.

## Reading Guide

This directory serves as the root for the site's main static pages. To understand the site's overall mission, start with `about.md`. For an overview of written content, visit `articles.md`. To explore practical applications and experiments, see `projects.md`. For curated tools and learning materials, refer to `resources.md`. Each page provides an introduction and links to deeper content within its respective category.
