# 02-articles

This directory contains the main articles and long-form content for the "Syntax & Empathy Companion" site, covering design leadership, AI collaboration, and technical workflows.

## Overview

The `02-articles` directory serves as the primary repository for in-depth articles, essays, and comprehensive guides that form the core intellectual content of the "Syntax & Empathy Companion." It provides foundational knowledge and perspectives on critical topics for design professionals, ranging from practical technical skills to broader leadership and ethical considerations. Visitors should explore this directory to find detailed discussions and insights relevant to enhancing their technical acumen, collaboration strategies, and leadership capabilities in the design and technology space.

## Structure

The subdirectories within `02-articles` are organized primarily by a chronological naming convention for published articles, followed by a functional directory for raw content. This structure allows for clear tracking of content development and easy identification of core articles.

**Naming Convention:** Article directories are prefixed with a reverse date format (YYMMDD), followed by a concise, descriptive slug. The `raw/` directory is a functional container for unprocessed content.

**Content Types:** Visitors will find in-depth articles, practical guides, conceptual essays, and thought leadership pieces. The `raw/` subdirectory specifically holds source content in various unformatted states.
