# Working Materials: Claude Code Preview Observational Study

This directory contains the draft article and all supporting raw data, session logs, and visual records used to document an observational study of Claude Code's preview version.

## Summary

This collection provides the foundational research and development artifacts for the `claude-code-preview.md` article, which details an observational study of Claude Code's preview functionality. Visitors will find raw conversational AI session logs (Claude and ChatGPT), a comprehensive briefing document for the core project, and visual evidence, all contributing to the analysis of AI collaboration in technical workflows. It is essential for understanding the empirical basis of the insights presented in the companion article.

## About This Content

This directory encapsulates the full scope of raw materials that informed the `claude-code-preview.md` article. The main theme explored is the practical application and performance of Claude Code's preview version in real-world development and debugging scenarios, specifically within an "AI/Human Text Detection Pipeline" project. The approach is observational, drawing directly from extensive session logs and visual records of interactions. Key insights revolve around AI's capabilities and limitations in complex coding tasks, iterative debugging processes, and the nuances of human-AI collaboration. The intended audience includes design technologists, AI researchers, and anyone interested in the practicalities of integrating large language models into technical workflows. This content provides the empirical backbone for the broader discussion on AI collaboration and structured technical documentation.

## Key Topics

-   **AI-assisted Development**: Real-world application of Claude Code for coding, debugging, and task execution.
-   **Human-AI Collaboration**: Analysis of interaction patterns, prompting strategies, and workflow integration with AI.
-   **Text Authenticity Analysis**: The underlying project context involving the development of an AI/Human text detection system.
-   **Observational Research**: Methodology for documenting and analyzing AI tool performance through direct interaction logs.
-   **Debugging Workflows**: Challenges and resolutions encountered during the development of complex computational notebooks with AI assistance.

## Content Structure

This directory contains:

-   **`claude-code-preview.md`** - The primary draft article documenting the observational study of Claude Code's preview version. This article synthesizes the findings from the various input materials present here.
-   **`ChatGPT-lexical-semantic orchestrator.md`** - A detailed log of a ChatGPT interaction, specifically an "Authenticity Analysis Workflow," providing comparative or complementary AI interaction context.
-   **`ai-v-human-brief.md`** - A briefing document outlining the development cycle and challenges of the `ai_v_human_v3` Jupyter notebook, which served as the main project context for the Claude interactions.
-   **`claude-session-log-X.md` (1-6)** - Extensive raw session logs detailing interactions with Claude AI, capturing prompts, responses, and system outputs during the observational study.
-   **`chat-gpt/`** - Contains further input materials related to the "Authenticity Analysis Workflow" involving ChatGPT.
-   **`task 1/`** - Houses input screenshots relevant to Task 1 of the Claude Code Preview analysis, providing visual context to specific interactions.
-   **`task 2/`** - Contains a visual log of interactions for Task 2 of the Claude Code Preview, offering additional graphical evidence.
-   **`transcripts/`** - Stores raw text transcripts derived from various interactions, serving as source material for the article.

## Reading Guide

To fully understand the context and findings, it is recommended to:
1.  **Start with `claude-code-preview.md`**: This is the main article draft, providing the synthesized observations and analysis.
2.  **Refer to `ai-v-human-brief.md`**: For a comprehensive understanding of the project that Claude AI was assisting with.
3.  **Explore `claude-session-log-X.md` files**: For a deep dive into the raw, chronological interactions with Claude. These logs provide the empirical evidence for the article's claims.
4.  **Review the `chat-gpt/`, `task 1/`, `task 2/`, and `transcripts/` subdirectories**: For specific visual and textual inputs that support particular sections or examples within the main article.
